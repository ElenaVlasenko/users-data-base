## Технологический стек
- TypeScript
- Webpack + npm
- React 16
- Ant Design 5 и styled-components
- React Router 6
- TanStack Query 4 + axios
- dayjs
- Git (GitHub)
- Архитектура: Feature-Sliced Design (FSD)

## Цель
- Реализовать на указанном стеке проект «База пользователей», содержащий основные компоненты:
	- Страница авторизации
	- Страница 404
	- Страница пользователей
	- Модальное окно редактирования пользователя
	- Модальное окно создания пользователя

## Описание проекта
- Приложение-демо «База пользователей», реализованное с использованием указанных технологий. Позволяет авторизоваться, просматривать список пользователей, создавать и редактировать пользователей через модальные окна, а также обрабатывать ошибки и навигацию на страницу 404.

# Руководство по запуску

## Установка
```bash
npm install
```
## Сервер разработки
```bash
npm run start
```
## Сборка для продакшна
```bash
npm run build
```
- Генерирует статические файлы в `dist/`

## Проверка типов
```bash
npm run typecheck
```
